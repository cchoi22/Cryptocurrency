# Cryptocurrency
## Overview
The overall objective of this exercise was to obtain data from a sample Cryptocurrency data set. Using the provided data set a unsupervised machine learning analysis would be conducted on the fields. A PCA model was used to first reduce and normalize the data. Subsequently a Kmean analysis was conducted on the reduced data set to understand how the data was clustered.

The data was reduced to 3 principle components and then ploted based on supply and demand. Most tradable coins fall under class 2 and 3 and have are similar in performance according to this analysis. This analysis did not take into to account the exchange rate to USD, just mined coin vs total supply remaining to be mined.
